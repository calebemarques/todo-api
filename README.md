# Todo API (FastAPI)

API de gerenciamento de tarefas desenvolvida com Python e FastAPI.

## Tecnologias
- Python
- FastAPI
- SQLAlchemy
- SQLite
- Uvicorn

## Funcionalidades
- Criar tarefas
- Listar tarefas
- Atualizar tarefas
- Deletar tarefas

## Como rodar

Clone o repositório:

git clone https://github.com/calebemarques/todo-api

Instale as dependências:

pip install -r requirements.txt

Execute:

uvicorn main:app --reload

Acesse:

http://127.0.0.1:8000/docs
